# FEC Proxy Server (Kathleen Hogan)

Proxy server for mock Grubhub project, serving all services from contributors.

### Usage

```
git clone https://github.com/hrla29-group-targaryen/proxy-Kathleen.git
cd proxy-Kathleen
npm install
npm run start
```
